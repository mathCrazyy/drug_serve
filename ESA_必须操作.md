# ⚠️ ESA 部署 - 必须手动操作

## 🎯 当前状态

✅ **构建成功** - 所有代码已正确构建
❌ **版本创建失败** - 需要手动删除旧版本

## 🔴 必须操作（无法通过代码解决）

这个问题**必须在 ESA 控制台手动操作**，无法通过代码或配置解决。

### 操作步骤（3 分钟完成）

1. **登录控制台**
   ```
   https://esa.console.aliyun.com/
   ```

2. **找到应用**
   - 左侧菜单：应用管理 → 应用列表
   - 找到：`drug_serve`
   - 点击进入应用详情

3. **进入版本管理**
   - 在应用详情页找到"版本管理"或"代码版本"标签
   - 点击进入版本列表

4. **删除旧版本**
   - 选择最早的或失败的版本
   - 点击"删除"按钮
   - **建议删除 5-10 个旧版本**（如果版本很多）

5. **重新构建**
   - 返回应用页面
   - 点击"构建"或"重新部署"
